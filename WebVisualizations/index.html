Effects of latitude on weather patterns

These visualizations were created to analyze how proximity to the equator affected local weather patterns. To begin, data from OpenWeatherMap API was pulled for a randomly generated list of over 500 cities.

From this dataset Matplotlib was utilized to pot latitude versus each of our four chosen markers of weather; maximum temperature, humidity, cloud coverage, and wind speed. You can find each of these individual visualizations on this site along with a page holding all four graphs for easier comparison. The source dataset was also included for reference.
